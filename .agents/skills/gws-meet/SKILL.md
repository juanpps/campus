# Google Workspace Meet API

Herramientas para interactuar con la API de Google Meet a fin de inyectar los enlaces generados directamente en los dashboards de los Tutores y Estudiantes de Método Raptor.

## Uso general
- Automatización en la creación de eventos concurrentes con conferencias ancladas.

REGLA DE AUDITORÍA DE SEGURIDAD:
**Prohibición MÁXIMA**: Queda prohibido el uso de Firebase Storage dentro de esta skill y en todo el proyecto. Ninguna grabación o archivo asociado se sube a Firebase.
